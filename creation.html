    How To Install

    You need to have cheats enabled in your world! Give yourself a command block with "/give @p command_block", and place it down. Copy the command above with the "Copy To Clipboard" button and paste it into the command block (with Ctrl + V or Cmd + V). After activating it, you will have the module installed.
